# 💪 AI-Powered Fitness Platform

A full-stack AI-powered fitness platform delivering **personalized workout and nutrition recommendations** using modern technologies and a **microservices architecture**.

---

## ✨ Features

- 🏋️ Personalized workout and nutrition plans powered by AI  
- 🤖 Real-time activity suggestions using the Google Gemini API with **92% prediction accuracy**  
- ⚡ Enterprise-grade microservices ecosystem for scalability, security & reliability  
- 🔗 Event-driven communication with Apache Kafka  
- 🔒 Secure authentication using Keycloak & OAuth2  
- 🐳 Dockerized deployment for easy setup and portability  

---

## 🛠 Tech Stack

- **Backend:** Spring Boot (Java), Apache Kafka, Keycloak, OAuth2, API Gateway, Eureka Server  
- **Frontend:** React.js  
- **Databases:** MongoDB, MySQL  
- **DevOps / Tools:** Docker, Postman, IntelliJ IDEA, Google API Key  

---

## 🏛 Architecture

- 🏗 Microservices architecture with service discovery using Eureka Server  
- 🌐 API Gateway for centralized routing & security  
- 🔄 Event-driven communication between services via Kafka  
- 🐳 Docker-based containerization for each service
